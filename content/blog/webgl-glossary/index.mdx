FragColor
The colour of the pixel being computed

FragCoord
The x and y value of the pixel being computed

Normalizing
Converting a value to be between 0.0 and 1.0
You can normalize the FragCoord by dividing it with by the canvas resolution

## Variables

Uniform
The same between pixels

Varying
Can be different between pixels
